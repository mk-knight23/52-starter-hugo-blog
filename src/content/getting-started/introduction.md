# Welcome to DOCS.

DOCS. is a modern documentation system built with Vue 3, Vite-SSG, and Tailwind CSS. It's designed to be fast, accessible, and easy to customize.

## Features

- **Static Site Generation** - Lightning-fast builds with Vite-SSG
- **Markdown Support** - Write documentation in Markdown with frontmatter
- **Search** - Full-text search across all documentation
- **Dark Mode** - System-aware theme switching
- **Responsive** - Mobile-first design that works everywhere
- **SEO Optimized** - Built-in meta tags, sitemaps, and RSS feeds

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Project Structure

```
src/
├── content/          # Markdown documentation files
├── components/       # Vue components
├── composables/      # Reusable composables
├── stores/          # Pinia state management
├── views/           # Page components
└── utils/           # Utility functions
```
